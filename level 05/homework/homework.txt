2. Html-ის დოკუმენტში კომენტარის სახით ჩამოწერეთ ყველა თეგი, რომელიც აქამდე ვისწავლეთ. თითოეულს მიუწერეთ დანიშნულება და
დააკონკრეტეთ Container tag-ია თუ Empty თეგი.


p tag - paragraph (<p></p>) - ამ თეგში იწერება ნორმალური ინფორმაცია, რაც საიტზე უნდა მოთავსდეს. არის Container tag,
რადგან აქვს გამხსნელი და დამხურავი თეგები.

h1-h6 tag - heading (<h1></h1>...<h6></h6>) - სათაურის თეგები. ამ თეგებში თავსდება სათაურები. h1, h2, h3, h4, h5, h6. 
h1 თეგი საიტზე უმნიშვნელოვანესია და თითო გვერდზე მხოლოდ ერთხელ უნდა იქნას გამოყენებული, ხოლო დანარჩენების გამოყენების
რაოდენობა არაა შეზღუდული. h1 ზომაში ყველაზე დიდია, ხოლო უკუ-მიმდევრობით პატარავდება და საბოლოოდ h6-ია ყველაზე მცირე ზომის. 
ექვსივე თეგი არის Container tag.

button tag - button (<button></button>) - ღილაკის თეგი. ამ თეგით იქმნება ღილაკი. არის Container tag.

br tag - line breaking (<br>) - ერთი ხაზით ჩამოსვლის თეგი. ამ თეგის დაწერისას, ელემენტი საიტზე ერთ ხაზით დაბლა იწერება. 
რაოდენობა შეუზღუდავია. შესაძლებელია თავად თეგებში ჩანესტვაც. არის Empty tag, ანუ აქვს მხოლოდ გამხსნელი თეგი.

img tag - image tag (<img>) - სურათის თეგი. ამ თეგის მეშვეობით საიტზე ფოტოს ვსვამთ. მას აქვს დამხმარე -  
src და alt ატრიბურები. შიგთავსი არა თეგში, არამედ ატრიბუტებში იწერება. 
<img src="address of the image" alt="explanation of image">

src - source (წყარო)
alt - alternative text (აღწერითი ტექსტი)

არის Empty tag.
